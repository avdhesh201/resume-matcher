# TalentMatch 🚀
> Helping recruiters find the perfect fit — fast, fairly, and accurately.

TalentMatch is a lightweight web app that ranks candidate resumes against a job description in seconds. Upload your files, set how many top matches you need, and instantly see candidates ranked by relevance score.

🌐 **Live Demo** → [talentmatch-t7u8.onrender.com](https://talentmatch-t7u8.onrender.com)

---

## ✨ Features

- 📄 Upload a Job Description (PDF, DOCX, or TXT)
- 👥 Upload multiple resumes at once
- ⚡ Instant ranking using TF-IDF + Cosine Similarity
- 🎯 Set your own Top-N threshold
---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | [Flask](https://flask.palletsprojects.com/) |
| Matching Engine | [scikit-learn](https://scikit-learn.org/stable/documentation.html) |
| PDF Parsing | [PyPDF2](https://pypdf2.readthedocs.io/) |
| DOCX Parsing | [python-docx](https://python-docx.readthedocs.io/) |
